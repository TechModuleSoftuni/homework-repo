-> Sugar content – **real number**, **suffixed** by “**g**” (e.g. “**30g**”) 

**Examples:**

**Input:**

->300

->70

660kcal, 154g sugars

250ml Ice Cold Nuka-Cola:

875kcal, 162.5g sugars

**Input:**

-> Nuka-Cola Quantum

-> 350

-> 600

-> 140 

**Output:**

350ml Nuka-Cola Quantum:

2100kcal, 490g sugars
